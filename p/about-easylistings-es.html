Connecting Your Data 🔌

Easy Listings is powered by your own Google Sheets, turning them into a simple, effective database. The setup is straightforward and gives you complete control over your content.

👤 User Database Setup

Manage user accounts and their status ("Waiting Approval", "Published", "Blocked") from a central Google Sheet.

View Setup Instructions

The application logic is ready to connect to a Google Sheet for user management. A backend script would be required to securely read this sheet and verify user credentials.

This process is currently mocked in the demo but is fully designed for a live production environment.

🏠 Listings Database Setup

Sync your property data directly from a Google Sheet by deploying a simple script.

View Script & Instructions
  1. Create a Google Sheet with your property data.
  2. Open `Extensions > Apps Script`.
  3. Paste the provided script code below.
  4. Deploy the script as a Web App.
  5. Set access permissions to "Anyone".
  6. Paste the generated URL into your settings.

API & Media Integration 🖼️

Streamline your media management by integrating with ImageKit.io for fast, optimized delivery of images, videos, and PDFs.

🚀 ImageKit.io for Media Storage & Rendering

Using ImageKit allows for direct uploads from the application, real-time image optimization, and reliable hosting for all your media files.

View ImageKit Setup Instructions
  1. Create a free account at ImageKit.io.
  2. In your dashboard, go to the "Developer" section in the sidebar.
  3. Under "API Keys", find and copy your Public API Key.
  4. Paste this key into the designated field in the application's Settings modal.
  5. For production, it is recommended to implement a secure, server-side authentication endpoint for uploads.
